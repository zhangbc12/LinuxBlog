搜索算法Ⅱ-Informed Search

Searching Problem的分类与定义

搜索的问题根据搜索的条件、限制和目标的不同，可以有以下几类定义:
1. Uninformed Search(盲目搜索)
2. Informed Search(知情搜索)
3. Local Search(局部搜索)

要解决搜索问题，首先需要明白如何去定义这个问题
1. States: 构成状态的详细信息
2. Initial state: 初始状态
3. Actions: 可以进行的操作
4. Transition: 状态的转移
5. Cost: 路径的成本函数
解决搜索问题的方案就是将初始状态转换为目标状态的一系列动作。


Informed Search

Informed Search(知情搜索)和Uninformed Search(盲目搜索)相比，能更有效地找到解决方案，因为Informed Search能通过Heuristic function(启发式函数)获得除问题本身之外的知识

Heuristic function是根据特定目标定义的一个函数，h(n)表示估计的(非实际的)第n个节点到目标节点的距离，目标节点处h(n)=0。


Greedy


贪心搜索Greedy，也叫最佳优先搜索，先探索h(n)最小的节点。


上图中，S为起始节点，G为目标节点，连线表示两点能够连通，需要解决的问题是找到从S到G的路径。

// Greedy
from heapq import heappush, heappop
def greedyGsa(stateSpaceGraph, h, startState, goalState): 
      frontier = []
      heappush(frontier, (h[startState], startState))
      exploredSet = set()
      print('Initial frontier:',list(frontier)); input()
      while frontier:
         node = heappop(frontier)
         if (node[1].endswith(goalState)): return node
         if node[1][-1] not in exploredSet:
            print('Exploring:',node[1][-1],'at cost',node[0])
            exploredSet.add(node[1][-1])
            for child in stateSpaceGraph[node[1][-1]]:
                  heappush(frontier, (h[child[1]], node[1]+child[1]))
            print(list(frontier)); print(exploredSet); input()
practice = {
      'S':[(3,'a'),(2,'d'),(10,'G')],'a':[(5,'b')],
      'd':[(1,'b'),(4,'e')],'G':[],'b':[(1,'e'),(2,'c')],
      'e':[(3,'G')],'c':[(4,'G')]}
practiceH = {'S':7,'a':9,'b':4,'c':2,'d':5,'e':3,'G':0}
print('Solution path:',greedyGsa(practice, practiceH, 'S', 'G'))

A*


A*算法是UCS和Greedy的一种结合，UCS关注最低的cost(n)的节点，Greedy关注最低的h(n)的节点，可以表示为f(n)=cost(n)+g(n)

// A*
from heapq import heappush, heappop  
def aStarGsa(stateSpaceGraph, h, startState, goalState): 
      frontier = []
      heappush(frontier, (h[startState], startState))
      exploredSet = set()
      print('Initial frontier:',list(frontier)); input()
      while frontier:
         node = heappop(frontier)
         if (node[1].endswith(goalState)): return node
         if node[1][-1] not in exploredSet:
            print('Exploring:',node[1][-1],'at cost',node[0])
            exploredSet.add(node[1][-1])
            for child in stateSpaceGraph[node[1][-1]]:
                  heappush(frontier, (node[0]+child[0]-h[node[1][-1]]+h[child[1]], node[1]+child[1]))
            print(list(frontier)); print(exploredSet); input()
practice = {
      'S':[(3,'a'),(2,'d'),(10,'G')],'a':[(5,'b')],
      'd':[(1,'b'),(4,'e')],'G':[],'b':[(1,'e'),(2,'c')],
      'e':[(3,'G')],'c':[(4,'G')]}
practiceH = {'S':7,'a':9,'b':4,'c':2,'d':5,'e':3,'G':0}
print('Solution path:',aStarGsa(practice, practiceH, 'S', 'G'))
